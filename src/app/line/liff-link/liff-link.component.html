การสร้าง Link เพื่อเข้าระบบ

Endpoint URL สำหรับ LIFF (แชร์สินค้ารายตัว)

{{ url }}{{ util.shopid }}&module=detail&liff=

1. เข้าไปยัง ลงทะเบียน

2. เลือก เข้าสู่ระบบด้วยบัญชี LINE

3. ทำการเข้าระบบด้วยบัญชี LINE (สามารถ เข้าสู่ระบบผ่าน QR Code ของ LINE ได้)

4. เลือก Create a new provider

5. ตั้งชื่อเป็น Remax แล้วกด Create

6. มาหน้าให้กดที่ Create a LINE Login channel

7. ตรง Channel icon ให้กด Register เพื่อทำการอัปรูปภาพ (ภาพนี้จะไปแสดงเมื่อได้ทำการแชร์สินค้าไปยังแพลทฟอร์มต่าง ๆ )

เช่นตรงนี้

8. ตรง Channel name ให้ใส่ชื่อของร้านตัวเอง (ความยามไม่เกิน 20 ตัวอักษร)
ส่วนตรง Channel description ให้ใส่ Remax เข้าไป
และตรง App types ให้เลือก Web app ตามในรูป

9. ตรง Email address ให่ใส่ อีเมลล์ ของช็อปหรือเจ้าของช็อปเข้าไป เพื่อรับแจ้งข่าวสารจากทางไลน์

10. ในส่วน Privacy policy URL , Terms of use URL สามารถข้ามไปได้ (ไม่ต้องใส่)
เสร็จแล้วติ๊ก i have read and agree (สามารถไปอ่านรายละเอียดก่อนก็ได้)
เสร็จแล้วให้กด Create เพื่อสร้าง LINE LOGIN นี้ขึ้นมา

11. เมื่อสร้างเสร็จแล้วจะมายังหน้านี้ ซึ่งจะมีแจ้งเตือนว่า Channel นี้ถูกสร้างมาแล้วนะ แล้วจะขึ้นรูปที่เราพึ่งอัปโหลดไป
พร้อมด้วย ชื่อ ที่พึ่งสร้างขึ้นมานั้นเอง

12. ทำการกดที่ Developing

13. จะขึ้นหน้าดังรูปขึ้นมา ให้เลือก Publish

14. ให้เลือกไปยังเมนู LIFF ตามรูป

15. ให้เลือก Add

16. จะมายังหน้าดังรูป

17. ให้กรอก Liff app name เป็นชื่อที่จะแสดงเมื่อ ช็อปได้ทำการแชร์สินค้า
Size ให้เลือกเป็น Full
Endpoint URL คืออันนี้ :

{{ url }}{{ util.shopid }}&module=detail&liff=


เอาไปใส่นะ

18. เลือก Profile , Openid , กด View All จะมี Chat_message.write ให้เลือก
ส่วน Bot link feature ให้เลือกเป็น On (Normal)

19. เสร็จแล้วให้กดที่ Add

20. จะเห็นว่า LIFF ได้ถูกสร้างขึ้นมาแล้ว ให้กดไปที่ LIFF ที่เราสร้างขึ้นตามภาพ

21. ตรงเส้นใต้แรกสุด จะเป็นค่า LIFF ID
ให้ทำการกด Edit ตรง Endpoint URL แล้วทำการนำ LIFF ID มาต่อท้าย ดังรูป

22. เสร็จแล้วจะได้ ตามรูปนี้นะ
เสร็จแล้วกดที่ Update

23. เสร็จแล้ว เอา LIFF URL จากข้อ 19
